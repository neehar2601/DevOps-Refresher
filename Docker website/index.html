What is Docker?

Docker is a platform for building, packaging, and running applications as containers. A container bundles your app + its dependencies, and runs in an isolated environment on a host.

Mental model: Image = blueprint (immutable), Container = running instance (mutable runtime layer).

Why containers?

Consistent runtime across laptops, CI, and production; quick startup; reproducible builds.

Consistency Speed Portability
